Task:
In this task, your goal is to create a simple web page that fetches data from a public API and displays it.

requirements:

- Select a public API that provides data of interest to you, you can choose whatever you like
    But here is one suggestion:
    JSONPlaceholder: A fake online REST API for testing and prototyping.
    https://jsonplaceholder.typicode.com/

    Pokemon api
    https://pokeapi.co/

- fetch the data using the fetch method in javascript

- Add a .catch to your fetch request, so that if the request fails, you display an error message to the user.
   
- Console.log the data from the API

- Display the data in a web page

    - Github repo
    - Netlify/vercel link
